# Feedback-U-Net

## Usage

1. Setup the dataset.

    Default path of ground truth dataset is ~/dataset/GroundTruth.
    
    Default path of input dataset is ~/dataset/Image.
    
1. Train the network.

    ```bash
    python3 main.py
    ```

1. Evaluate the network.

    ```bash
    python3 test.py
    ```
